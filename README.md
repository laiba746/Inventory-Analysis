# Inventory-Analysis
Data analysis project using Excel Pivot Tables to track inventory performance, compare purchased vs sold quantities, and generate insights for stock management decisions.
📊 Project Overview

This project focuses on analyzing inventory data using Excel Pivot Tables to gain insights into product performance, stock movement, and sales efficiency. The analysis helps in understanding how products are being purchased, sold, and stored over time.

 Objectives
Compare purchased stock vs sold units for each product
Analyze sales performance and demand patterns
Calculate sales ratio and remaining stock
Visualize data using charts for better insights
 Dataset Description

The dataset contains product-level inventory information including:

Product ID & Product Name
Opening Stock
Purchase/Stock In
Number of Units Sold
Cost Price per Unit
Total Cost Value
                 Key Analysis Performed
1. Purchased vs Sold Comparison
Compared total purchased stock with units sold
Identified high-demand and low-demand products
2. Sales Ratio (%)
Calculated sales efficiency using:
Units Sold / Total Available Stock
Used to measure product performance
3. Remaining Stock Analysis
Calculated remaining inventory after sales
Helps in stock management decisions
4. Inventory Value Insights
Evaluated product-wise stock value using cost price
